# Login Web App

A simple Login web application built with **Node.js**, **Express.js**, and **MongoDB Atlas**. Users can log in with their registered email and password and view their profile.

## Features

- User login with email and password
- Profile details display
- MongoDB Atlas integration
- Clean and simple UI

## Technologies

- Node.js
- Express.js
- MongoDB Atlas
- HTML, CSS
